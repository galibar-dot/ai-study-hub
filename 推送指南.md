# AI Study Hub - 推送到 GitHub/Gitee 完整指南

## 🎯 三步完成推送

---

## 第一步：配置 Git 用户信息（只需做一次）

打开 PowerShell 或命令提示符，执行以下命令：

```powershell
# 配置你的名字（用于提交记录）
git config --global user.name "你的名字"

# 配置你的邮箱
git config --global user.email "你的邮箱@example.com"
```

**示例**：
```powershell
git config --global user.name "Zhang San"
git config --global user.email "zhangsan@example.com"
```

---

## 第二步：提交代码到本地仓库

```powershell
# 进入项目目录
cd "claude code1"

# 提交代码
git commit -m "Initial commit: AI Study Hub v1.0.0"
```

---

## 第三步：推送到远程仓库

### 选项 A：推送到 GitHub（国际）

#### 1. 创建 GitHub 仓库
1. 打开浏览器，访问：https://github.com/new
2. 填写信息：
   - **Repository name**: `ai-study-hub`
   - **Description**: `AI-powered learning platform with intelligent chat, English learning and reading comprehension`
   - **Public** 或 **Private**：选择 Public（公开）
   - ⚠️ **不要勾选** "Add a README file"
3. 点击 **Create repository**

#### 2. 关联并推送
复制你的 GitHub 用户名，然后执行：

```powershell
# 关联远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/ai-study-hub.git

# 推送代码
git branch -M main
git push -u origin main
```

**示例**（假设你的用户名是 zhangsan）：
```powershell
git remote add origin https://github.com/zhangsan/ai-study-hub.git
git branch -M main
git push -u origin main
```

#### 3. 输入凭据
推送时会要求输入：
- **Username**: 你的 GitHub 用户名
- **Password**: 你的 Personal Access Token（不是密码！）

**如何获取 Personal Access Token**：
1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. 勾选 **repo** 权限
4. 点击 **Generate token**
5. 复制生成的 token（只显示一次，请保存好）

---

### 选项 B：推送到 Gitee（国内推荐，速度快）

#### 1. 创建 Gitee 仓库
1. 打开浏览器，访问：https://gitee.com/projects/new
2. 填写信息：
   - **仓库名称**: `ai-study-hub`
   - **仓库介绍**: `AI 驱动的学习平台，集成智能对话、英语学习和阅读训练功能`
   - **是否开源**：选择 **公开**
   - ⚠️ **不要勾选** "使用 Readme 文件初始化这个仓库"
3. 点击 **创建**

#### 2. 关联并推送
复制你的 Gitee 用户名，然后执行：

```powershell
# 关联远程仓库（替换 YOUR_USERNAME 为你的 Gitee 用户名）
git remote add origin https://gitee.com/YOUR_USERNAME/ai-study-hub.git

# 推送代码
git push -u origin master
```

**示例**（假设你的用户名是 zhangsan）：
```powershell
git remote add origin https://gitee.com/zhangsan/ai-study-hub.git
git push -u origin master
```

#### 3. 输入凭据
推送时会要求输入：
- **Username**: 你的 Gitee 用户名
- **Password**: 你的 Gitee 密码

## ❓ 常见问题

### Q1: 提示 "Author identity unknown"
**A**: 需要先配置 Git 用户信息（见第一步）

### Q2: 提示 "remote: Repository not found"
**A**: 确保你已经在 GitHub/Gitee 上创建了仓库，并且用户名正确

### Q3: 提示 "remote origin already exists"
**A**: 说明已经关联过了，跳过关联步骤，直接推送：
```powershell
git push -u origin main  # GitHub
# 或
git push -u origin master  # Gitee
```

### Q4: GitHub 推送时密码不对
**A**: GitHub 不再支持密码登录，必须使用 Personal Access Token

### Q5: 推送速度很慢
**A**: 国内用户推荐使用 Gitee，速度更快

## 📋 开源前最后检查

### 第一步：清理敏感信息
```bash
# 1. 检查是否有真实密码
grep -r "password\|api_key\|secret" *.py *.yml

# 2. 检查配置文件
cat config.yml
cat .env

# 3. 确认 .gitignore 正确
git status
```

### 第二步：清理用户数据
```bash
# 删除所有用户数据
rm -rf chats/* logs/* __pycache__
rm -f *.db  # 除了 ecdict.db 的其他数据库
```

### 第三步：测试安装流程
```bash
# 1. 克隆到新目录测试
git clone <your-repo> test-install
cd test-install

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境
cp .env.example .env
cp config.yml.example config.yml

# 4. 运行测试
python server.py
```

### 第四步：创建 Git 仓库
```bash
# 1. 初始化仓库
git init

# 2. 添加文件
git add .

# 3. 检查将要提交的文件
git status

# 4. 确认没有敏感文件
git diff --cached

# 5. 提交
git commit -m "Initial commit"

# 6. 推送到 GitHub
git remote add origin <your-repo-url>
git push -u origin main
```

```bash

rm -rf chats logs __pycache__
rm -f config.yml .env
```

## 💡 使用流程

### 首次使用（3步）

1. **启动管理器**
   ```
   双击：manager.bat
   ```

2. **启动服务器**
   ```
   输入：1
   按回车
   等待启动完成
   ```

3. **打开聊天**
   ```
   输入：6
   按回车
   ```

### 日常使用

**快速启动**：
```
manager.bat → 1 → 6
(启动管理器 → 启动服务器 → 打开聊天)
```

**修改配置后重启**：
```
manager.bat → 3
(启动管理器 → 重启服务器)
```

**临时关闭**：
```
manager.bat → 4 → 0
(启动管理器 → 停止服务器 → 退出)
```

## ⚠️ 重要提示

### 1. 首次使用需要安装依赖

已自动安装 `requests` 库，如果遇到问题：
```bash
pip install requests
```

### 2. 重启和停止功能需要登录

**步骤**：
1. 浏览器打开：http://localhost:8000
2. 输入密码：130481
3. 登录后即可使用重启/停止功能

### 3. 服务器窗口不要关闭

- 启动服务器会打开新窗口
- 不要手动关闭该窗口
- 使用管理器的"停止服务器"功能

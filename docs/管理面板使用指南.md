# 管理面板使用指南

## 🎯 快速开始

### 方式 1：双击启动（推荐）

1. **启动服务器**
   - 双击 `start.bat`
   - 等待服务器启动完成

2. **打开管理面板**
   - 双击 `admin.bat`
   - 或手动访问：http://localhost:8000/admin.html

### 方式 2：命令行启动

```bash
# 启动服务器
python server.py

# 打开管理面板
start http://localhost:8000/admin.html
```

---

## 🎨 管理面板功能

### 1. 服务状态监控

**实时显示**：
- 🟢 服务状态（运行中/已停止）
- 📍 本地地址（http://localhost:8000）
- 🌐 隧道状态（已连接/未连接）

**自动刷新**：
- 每30秒自动检查服务状态
- 手动点击 "🔄 刷新状态" 立即更新

---

### 2. 快捷操作按钮

#### 🌐 打开本地
- 在新标签页打开聊天界面
- 快速访问主应用

#### 🔄 刷新状态
- 手动检查服务器状态
- 更新状态显示

#### ⚡ 重启服务
- 一键重启服务器
- 自动等待5秒后检查状态
- 适用场景：
  - 修改配置后需要重启
  - 服务出现异常
  - 更新代码后重启

#### ⏹️ 停止服务
- 停止服务器运行
- 需要手动重新启动
- 适用场景：
  - 临时关闭服务
  - 系统维护

#### 🚀 启动 Cloudflare 隧道
- 提示手动运行隧道脚本
- 隧道脚本位置：
  - `tunnel.bat` (Windows批处理)
  - `tunnel.ps1` (PowerShell)

---

### 3. 操作日志

**实时显示**：
- 所有操作的执行记录
- 带时间戳的日志
- 颜色区分：
  - 🔵 蓝色 - 信息
  - 🟢 绿色 - 成功
  - 🔴 红色 - 错误

**日志管理**：
- 点击 "清空" 按钮清除日志
- 自动滚动到最新日志

## 🔧 API 接口说明

管理面板使用以下API接口：

### 健康检查
```http
GET /api/health
```
- 无需认证
- 返回服务器状态

### 重启服务器
```http
POST /api/admin/restart
```
- 需要登录认证
- 服务器将在1秒后重启

### 停止服务器
```http
POST /api/admin/stop
```
- 需要登录认证
- 服务器将在1秒后停止

## 📱 移动端访问

### 同WiFi访问

1. **查看电脑IP**
   ```bash
   ipconfig
   ```
   找到 IPv4 地址，例如：192.168.1.100

2. **手机访问**
   ```
   http://192.168.1.100:8000/admin.html
   ```

### 外网访问（通过隧道）

1. **启动隧道**
   - 双击 `tunnel.bat`
   - 或运行 `tunnel.ps1`

2. **获取隧道地址**
   - 查看隧道输出的URL
   - 例如：https://xxxx.trycloudflare.com

3. **访问管理面板**
   ```
   https://xxxx.trycloudflare.com/admin.html
   ```

## 🎯 使用场景

### 场景 1：日常使用

1. 双击 `start.bat` 启动服务器
2. 双击 `admin.bat` 打开管理面板
3. 点击 "🌐 打开本地" 使用聊天功能

### 场景 2：修改配置后重启

1. 修改 `config.yml` 或其他配置
2. 打开管理面板
3. 点击 "⚡ 重启服务"
4. 等待5秒后自动检查状态

### 场景 3：外网访问

1. 启动服务器
2. 运行 `tunnel.bat` 启动隧道
3. 在管理面板查看隧道状态
4. 使用隧道地址访问

### 场景 4：系统维护

1. 打开管理面板
2. 点击 "⏹️ 停止服务"
3. 进行系统维护
4. 完成后双击 `start.bat` 重新启动

## ⚠️ 注意事项

### 1. 重启功能

**Windows系统**：
- 重启会打开新的命令行窗口
- 旧窗口会自动关闭
- 如果重启失败，请手动重启

**限制**：
- 重启功能依赖系统权限
- 某些情况下可能需要手动重启

### 2. 停止功能

**警告**：
- 停止后需要手动重新启动
- 所有连接会断开
- 未保存的数据可能丢失

### 3. 隧道功能

**当前状态**：
- 需要手动运行隧道脚本
- 管理面板只提供提示
- 未来版本将支持一键启动

### 4. 安全性

**建议**：
- 管理面板需要登录认证
- 不要在公网暴露管理面板
- 定期更改登录密码

## 🐛 故障排查

### 问题 1：管理面板打不开

**原因**：服务器未启动

**解决**：
1. 检查服务器是否运行
2. 双击 `start.bat` 启动服务器
3. 等待启动完成后再访问

### 问题 2：重启按钮没反应

**原因**：
- 未登录
- 网络连接问题
- 权限不足

**解决**：
1. 确认已登录主应用
2. 检查浏览器控制台错误
3. 尝试手动重启服务器

### 问题 3：状态显示"已停止"但服务器在运行

**原因**：
- 网络延迟
- 防火墙阻止

**解决**：
1. 点击 "🔄 刷新状态"
2. 检查防火墙设置
3. 尝试访问主应用确认

### 问题 4：隧道无法启动

**原因**：
- cloudflared.exe 不存在
- 网络问题

**解决**：
1. 确认 cloudflared.exe 在项目根目录
2. 检查网络连接
3. 查看隧道脚本输出的错误信息

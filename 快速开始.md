# AI Study Hub - 快速开始指南

## 🚀 立即推送到 GitHub/Gitee

### 方法一：使用自动化脚本（推荐）

```powershell
# 在项目目录运行
.\推送到远程仓库.ps1
```

脚本会自动：
1. ✅ 提交代码
2. ✅ 选择平台（GitHub/Gitee）
3. ✅ 关联远程仓库
4. ✅ 推送代码

### 方法二：手动执行命令

#### 推送到 GitHub

```powershell
# 1. 提交代码
git commit -m "Initial commit: AI Study Hub v1.0.0"

# 2. 在浏览器创建 GitHub 仓库
# 访问: https://github.com/new
# 仓库名: ai-study-hub
# 描述: AI-powered learning platform with intelligent chat, English learning and reading comprehension
# 公开/私有: Public
# 不要勾选 "Initialize this repository with a README"

# 3. 关联远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/ai-study-hub.git

# 4. 推送代码
git branch -M main
git push -u origin main
```

#### 推送到 Gitee（国内推荐）

```powershell
# 1. 提交代码
git commit -m "Initial commit: AI Study Hub v1.0.0"

# 2. 在浏览器创建 Gitee 仓库
# 访问: https://gitee.com/projects/new
# 仓库名: ai-study-hub
# 描述: AI 驱动的学习平台，集成智能对话、英语学习和阅读训练功能
# 公开/私有: 公开
# 不要勾选 "使用 Readme 文件初始化这个仓库"

# 3. 关联远程仓库（替换 YOUR_USERNAME）
git remote add origin https://gitee.com/YOUR_USERNAME/ai-study-hub.git

# 4. 推送代码
git push -u origin master
```

## 🔧 常见问题

### Q: 推送时提示 "remote: Repository not found"
**A:** 确保你已经在 GitHub/Gitee 上创建了仓库，并且仓库名正确。

### Q: 推送时需要输入用户名和密码
**A:** 
- GitHub: 使用 Personal Access Token 代替密码
- Gitee: 使用账号密码或 SSH 密钥

### Q: 如何生成 GitHub Personal Access Token？
**A:** 
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 "repo" 权限
4. 生成并保存 token
5. 推送时使用 token 作为密码

### Q: 推送速度很慢怎么办？
**A:** 
- 国内用户推荐使用 Gitee
- 或使用 GitHub 镜像加速
- 或配置 Git 代理
